# NotAFan 👻

Don't you hate it when someone unfollows you but keeps you following them? 
Like they want the number but not the relationship. Who do they think they 
are — do they think you're a fan?

**NotAFan** is a Chrome extension that shows you everyone you follow on 
Instagram who doesn't follow you back. No sketchy apps, no handing over 
your password to some shady website. Runs entirely in your browser using 
your own session.

## Install

1. Download the zip and unzip it  
2. Go to `chrome://extensions` and turn on **Developer mode** (top right)  
3. Click **Load unpacked** and select the unzipped folder  
4. Open instagram.com (logged in), click the extension icon, hit **Scan**

## How it works

Uses the same API Instagram's own website calls — just diffs your following 
vs followers list locally. Nothing leaves your browser.

## Notes

- If Instagram was already open when you installed it, refresh that tab once before scanning  
- Large accounts take longer — intentional, aggressive scanning can get your account flagged  
- Uses Instagram's internal API so it could break if they update things
